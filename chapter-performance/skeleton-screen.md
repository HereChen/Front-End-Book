# 骨架屏/Skeleton Screen

骨架屏指的是数据呈现之前, 一般用浅色的色条勾勒渲染后的轮廓. 相对通常的空白区域或者加 loading, 体验会好一些. 其次还起到了占位的作用.

## 实例

Ant Design 的 loading card, <https://ant.design/components/card/>
