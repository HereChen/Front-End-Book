# Front-End-Book

```bash
# npm install gitbook-cli -g

# 本地查看
gitbook serve

# HTML
gitbook build ./ ./docs

# PDF
# 需要安装 calibre, https://calibre-ebook.com
# Ubuntu 安装: sudo apt install calibre
gitbook pdf ./ ./Front-End-Book.pdf

# epub
gitbook epub ./ ./Front-End-Book.epub

# mobi
gitbook mobi ./ ./Front-End-Book.mobi
```
