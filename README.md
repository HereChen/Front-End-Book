# Front-End-Book

目录见 [SUMMARY.md](SUMMARY.md)。

```bash
# npm install gitbook-cli -g

# 本地查看
gitbook serve

# HTML
gitbook build ./ ./docs

# PDF
# 需要安装 calibre, https://calibre-ebook.com
# Ubuntu 安装: sudo apt install calibre
gitbook pdf ./ ./mybook.pdf

# epub
gitbook epub ./ ./mybook.epub

# mobi
gitbook mobi ./ ./mybook.mobi
```
