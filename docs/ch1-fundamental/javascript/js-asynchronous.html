异步

异步

异步的方法

  1. 回调, 适应于异步处理较少的情况

     get(url, callback);

  2. Promise, 链式方式写异步处理

     promise(somthing).then(res => handle)

  3. async 和 await, 同步的方式写异步, 可适用于异步处理之间存在依赖的情况.

     const promiseFunc = async () => {
   const result1 = await getSomething();
   const result2 = await dependOnResult1(result1);
   return result2;
 };
 promiseFunc().then(result => {
   console.log(result);
 })
