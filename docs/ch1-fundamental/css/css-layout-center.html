垂直居中和水平居中

垂直居中和水平居中

flex 也可用于垂直水平居中

块元素垂直居中(transform)

此方案存在兼容性问题(Firefox 43 不支持, Chrome 46 支持)

{
  width: 250px;
  height: 250px;
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}

块元素垂直居中(绝对定位)

{
  width: 250px;
  height: 250px;
  position: absolute;
  top: 50%;
  margin-top: -125px;
}

内联元素垂直居中

{
  /* 方案一: 块元素内容会居中, 需要设置高度 */
  display: table-cell;
  vertical-align: middle;

  /* 方案二: 一般单行文本的上下居中 */
  line-height: 50px;
}

块元素的水平居中

{
  width: 100px;
  margin-left: auto;
  margin-right: auto;
}
