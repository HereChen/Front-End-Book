# 体验优化

对页面性能的优化算起来都是体验优化, 这里主要指具有进一步提升性质的. 比如, 骨架屏实际上也可以用转圈圈来替代, 但其使用感受更好.

## 骨架屏/Skeleton Screen

骨架屏指的是数据呈现之前, 一般用浅色的色条勾勒渲染后的轮廓. 相对通常的空白区域或者加 loading, 体验会好一些. 其次还起到了占位的作用.
